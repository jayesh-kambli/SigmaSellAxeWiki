# SigmaSellAxe Wiki

Welcome to the complete documentation for **SigmaSellAxe** - An advanced container selling tool for Minecraft servers.

## ðŸŽ¯ What is SigmaSellAxe?

SigmaSellAxe transforms your server's economy by allowing players to instantly sell items from containers (chests, barrels, shulkers) using a special Sell Axe tool. It features automatic shop plugin integration, protection plugin support, and comprehensive transaction logging.

## âœ¨ Key Features

- **Instant Container Selling** - Right-click any chest, barrel, or shulker box to sell all contents
- **Multi-Shop Support** - ExcellentShop & EconomyShopGUI integration with automatic detection
- **Protection Plugin Support** - Lands, GriefPrevention, and WorldGuard integration
- **Smart Configuration** - Auto-generates shop configs based on detected shops
- **Duplication Exploit Detection** - Advanced system to detect potential money duplication
- **GUI Sorting** - Beautiful GUI with 4 sorting types for sellable items
- **Transaction Logging** - Detailed logging to console and files
- **Folia Compatible** - Fully supports Folia servers
